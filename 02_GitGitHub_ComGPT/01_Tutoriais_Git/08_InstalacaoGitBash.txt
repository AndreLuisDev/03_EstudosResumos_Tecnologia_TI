# Como instalar o Git Bash no Windows

 Instalação do Git Bash no Windows – Passo a Passo
Acesse o site oficial
https://git-scm.com

Clique em “Download for Windows”
Ele detecta seu sistema automaticamente e baixa o instalador .exe.

Execute o instalador (Git-x.x.x-64-bit.exe)

Siga o assistente de instalação, mantendo as opções padrão:

Editor padrão: normalmente vem como Vim (pode mudar para Notepad++ ou outro se preferir).

Git no PATH: escolha a opção "Git from the command line and also from 3rd-party software" (recomendado).

Linha de comando usada: selecione “Use MinTTY (the default terminal of MSYS2)” (que é o Git Bash que você vê).

Opções de CRLF: mantenha o padrão “Checkout Windows-style, commit Unix-style line endings”.

Clique em “Install”
Espere finalizar e clique em “Finish”, com a opção de abrir o Git Bash já marcada (se quiser).
